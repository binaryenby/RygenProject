Instructions for running the app:

1. Install Node.js if it isn't already installed
2. Open a command prompt in the 'RygenProject' folder
3. Type 'npm install -g serve' into the command prompt and hit enter
4. After Node.js is finished installing serve, type 'serve -s build' and hit enter
5. A window will be displayed in the command prompt telling you what port the app is currently running on. 
		Just type the localhost address it provides into the address bar of your browser
